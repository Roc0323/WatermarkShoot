## WatermarkShoot
拍照、拍摄小视频添加水印

***注：因为项目中需要实现拍摄照片、视频添加水印的需求，在网上找了一圈都没找到特别好用的三方，于是自己借鉴其他库实现了一个拍摄照片及视频添加水印的功能，本项目中的水印均以图片的形式添加进去！！！***
